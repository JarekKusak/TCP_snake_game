TCP Snake Game - Dokumentace

TCP Snake Game je multiplayerová hra Had běžící přes síťový protokol TCP. Hráči se připojují k serveru jako klienti a ovládají svého hada pomocí klávesnice.

Architektura

Hra se skládá z několika hlavních tříd:

  • Server – Spouští herní server a spravuje klienty.
  • Client – Připojuje se k serveru a zobrazuje herní stav.
  • Player – Spravuje pohyb hráčů a kolize.
  • Common – Obsahuje konstanty sdílené mezi klientem a serverem.
  • Position – Reprezentuje pozici hráče na mapě.

Funkcionalita

Hráči sbírají jablka pro body, vyhýbají se kolizím a mohou využít power-upy. Hra probíhá v kolech a poslední přeživší získává bonusové body.
